# Project Goals: RR Institute Karnal Website

## Overview
A professional website for **RR Institute Karnal**, an immigration and visa consulting firm. Inspired by premium immigration themes like Imigo.

## Primary Goals
- Create a visually stunning, premium-feel website.
- Showcase services: Study Visa, Tourist Visa, PR, IELTS/PTE Coaching.
- Generate leads via consultation and inquiry forms.
- High performance and SEO optimization.

## Key Features
- Responsive design (Mobile-first).
- Interactive service and country cards.
- Testimonial carousel and success stories.
- Professional branding (Navy Blue, Red, Gold/Yellow palette).

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Planned for animations)
